2D Multiplayer Game

https://craft-io.herokuapp.com/

I made this game as an honors contract project for my 2D game programming class.

I really wanted to start making socketIO games, so I made this as practice so I could go on to make cooler games. The initial idea was a Minecraft/Minicraft 2D multiplayer game where you could break trees, craft wood/tools, and build forts while playing with others. However I only added a few features since it proved to be very difficult to add new features.

You start by typing your name, then you can walk around with WASD, use numbers 1-9 to change inventory slots. press space to place items / pick up items / or punch another player.

This is the starting state. Asks you to enter your name, then the game starts.

Multiple different players gathering in one spot on the map.

When you punch another player they die and confetti shoots everywhere.
