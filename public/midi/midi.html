MIDI Instrument Replacer

github

This program takes a midi song file, and a .wav sound effect, and generates a .wav song using that sound effect. If you've seen those ( Call Me Maybe but its Gabe the Dog ) or ( Fireflies but with an Airhorn ) videos, thats what this program does.

I had to do a lot of research of .midi files and how to extract information from them. I ended up using a ton of Python libraries to make this easier. It's been a while since I made this, but I'm pretty sure a midi file has a list of notes, each note object containing a Time ( time since start of song ), Intensity ( volume ), and pitch. And there can be multiple tracks for one midi song, such as vocal track, and instrumantal track. So I looped through all the note objects in all of the tracks.

For the provided sound effect I assume that it is already pitched to a middle C. Then I create a chromatic scale by pitch shifting that sound effect up and down by half steps. I label them all so I can easily reference them in the future.

I create an empty song object where I append notes to as I loop through the note objects. I pull the corresponding sound effect for each note by looking at the notes pitch. Once I have looped through all notes, I export the song.

One big issue with this is that one of the libraries I use only works on Linux. So its bests to use this on Ubuntu.
