Ember.Js Support App


My Freshman year I started working with Dr. James Hill at IUPUI. He had students do full stack development projects for real clients. The project I worked on was a program that would allow life coaches and researchers interact with inmates coming out of incarceration. Inmates would be assigned life coaches where they would report to by creating "Goals", reporting "Distributions", and other features.

We were provided with UI designs of each page which I used to create the front ends for several pages using Ember.js.

Dr. Hill has created his own Node.js framework called "Blueprint", we used Blueprint and Mongodb for the backend.

I was very inexperienced when starting working on this project, but I learned a lot during my time on it. Unfortunately our client decided they couldn't use the application anymore as we were nearing completing the application.
