AP Computer Science Final Project

Download Zip

I made this Java game for my final project in my AP Computer Science class my Senior year of High School.

I made a decent number of Java games in high school and a couple Minecraft mods, so I knew the process fairly well. However it's been almost 4 years since I made this so I can't remember much about it.

I spent a long time making this, and I made a "Dev" version where I could easily add collision boxes. This saved time having to manually enter the coordinates for each collision box. However I'm sure there could have been an easier way to do this.

The plot of the game is you can play as "Ben", or "Jon" ( my 2 partners for this project who didn't contribute anything. ). You then talk to Haiwe ( our friend ) and walk around collecting lunch money. You have to avoid the assistant principal because you don't have an ID and you'll get in trouble ( have to start over ). Once you collect all the dollars I'm not sure what happens.
