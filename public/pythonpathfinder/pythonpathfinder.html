Class Pathfinder ( Python )

Download Zip

I created this for CSCI-230 ( Introduction to Computer Science ) class final project. I was living at the Tower on campus at the time, and I would typically sleep until I had just enough time to roll out of bed and walk to class. So I wanted a program that would tell me exactly how long it would take to get to class, and what the fastest way would be.

At the time I wasn’t experienced enough to create my own implementation of the A* pathfinding algorithm ( I was a Freshman ), So I used a python library that would take a 2d array of obstacles, a starting point, and an ending point. It would then return a list of points where the path would be.

I created the interface with TKinter, the background was a screenshot of IUPUI’s Campus map. I created a “Dev” mode where you could left click to add obstacles ( like fill in the borders of buildings ) and right click to delete obstacles. I just created a 2d grid of red squares and took the location of the mouse click and divided by the length/width of the squares to find out which one the user clicked on. I also made a way to save your configuration to a text file which would be loaded when the program starts.

Finally to use the program just left click to select the start position, and right click to select the end position, then click “Path”.
