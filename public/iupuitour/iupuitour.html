IUPUI Class Finder 3D Game

Play

I made this for my 3D Game Programming honors contract project. This game is supposed to help new students familiarize themselves with IUPUI's campus so they know how to get to their classes on their first day. They should start the game, then select their class buildings in the order that they would go to them in a day. Then it drops them by the Campus Center and they have to navigate to each of their classes. Everytime they walk by a building it will show the name to them. Everytime they reach the next class on their schedule, it will be removed from their schedule. Once they complete their schedule they can quit or restart. I used the skills I learned from my "Quarantine Simulator" game to quickly make this.

I use the same Blender OSM add on to make a 3D model of the IUPUI area. I then manually textured the buildings, roads, and ground. I then used Godot to make a main menu, win screen, and first person controller to walk around IUPUI.

I designed this so it would be easy for me to add new buildings options to the game. All I have to do is create a new Area object and place it on a building, name it, then that building will automatically be added to the main menu buildings list.

The only thing left to do with this project is improve the textures.
